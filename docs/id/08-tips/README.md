# 💡 08. Tips dari Trader Berpengalaman & Anti-Scam

> 🎯 **"The goal is not to get rich quick. The goal is to not lose money while learning."** — wisdom trader senior

## 🧠 Mindset (Paling Penting)

### 1. Crypto Bukan Jalan Pintas Kaya
- 90% trader **rugi** dalam 6 bulan pertama
- Butuh waktu 1–2 tahun untuk konsisten profit
- Treat sebagai **belajar**, bukan "cari uang cepat"

### 2. Aturan 24 Jam
Setelah FOMO mau buy, **tunggu 24 jam**. Kalau masih merasa perlu, baru trade dengan modal kecil.

### 3. Journaling
Catat setiap trade:
- Entry reason
- Exit reason
- P&L
- Lesson learned

Contoh template:
```markdown
### Trade 2026-07-05
- Token: $PEPE
- Entry: $0.00001 (MC $50K)
- Exit: $0.00002 (MC $100K)
- P&L: +95%
- Reason entry: trending di Twitter, holders naik
- Reason exit: take profit 2x
- Lesson: timing penting, exit lebih awal tidak apa-apa
```

### 4. Risk Management > Profit
Trader sukses bukan yang paling sering profit, tapi yang **paling jarang rug** besar.

## 🚫 Kesalahan Fatal yang Harus Dihindari

### ❌ 1. All-In Satu Token
```
Modal: $1000 → all-in $TOKEN_X → rug → $0
```
✅ Solusi: max 5% per token, diversifikasi 10+ token.

### ❌ 2. Revenge Trading
```
Loss $200 → emosi → all-in token lain → loss lagi → $0
```
✅ Solusi: setelah loss besar, **stop trading 24–48 jam**.

### ❌ 3. FOMO di Puncak
```
Token naik 100x → "mungkin masih naik 10x" → buy → dump 90%
```
✅ Solusi: kalau Anda baru tahu token-nya, **sudah terlambat**.

### ❌ 4. Percaya "Insider Info"
```
Grup Telegram "alpha" bilang token X akan pump 1000x → buy → rug
```
✅ Solusi: insider tip di Telegram = **99% scam**. Bahkan kalau real, Anda jadi **exit liquidity**.

### ❌ 5. Tidak Pakai Stop Loss
```
"Token ini pasti naik lagi" → -50% → -80% → -99%
```
✅ Solusi: stop loss **wajib**, pakai mental stop (catat di journal).

### ❌ 6. Pinjam Uang untuk Trade
```
Pinjam $10K → trade → loss → utang → stress
```
✅ Solusi: hanya trade uang yang siap hilang 100%.

## 🕵️ Skema Scam yang Sering Terjadi

### 1. 🚨 Rug Pull (Tarik Karpet)
```
Dev launch token → banyak beli → MC naik → dev jual semua → MC → 0
```
**Hindari:**
- Cek LP lock
- Hindari token tanpa audit
- Pantau dev wallet (kalau dev jual, Anda harus jual juga)

### 2. 🍯 Honeypot
```
Bisa beli token → tidak bisa jual (logic di smart contract)
```
**Hindari:**
- Test buy & sell kecil dulu
- Cek apakah ada blacklist function
- Lihat di honeypot.is

### 3. 🐋 Whale Dump
```
1 whale pegang 30% supply → jual pelan-pelan → harga turun
```
**Hindari:**
- Cek top holders concentration
- Hindari token dengan top 10 > 50%

### 4. 💸 Buy Tax / Sell Tax Tinggi
```
Buy 5% tax, sell 99% tax → dev bisa set tax arbitrary
```
**Hindari:**
- Cek tax di DexScreener atau Birdeye sebelum buy
- Normal: 0–5% tax. Bahaya: > 10%

### 5. 🎭 Fake Partnerships
```
"Akan listing di Binance besok" → tidak ada listing → dump
```
**Hindari:**
- Cek pengumuman resmi di website exchange
- Jangan percaya rumor tanpa bukti

### 6. 🤖 Pump & Dump Group
```
1000 anggota grup → "buy sekarang!" → 1 orang sudah beli duluan → dump ke anggota
```
**Hindari:**
- Kalau grup full strangers, **exit liquidity** Anda
- Smart money tidak perlu share alpha

### 7. 🐛 Approve Exploit
```
Connect wallet → approve unlimited spend → drain semua token Anda
```
**Hindari:**
- Selalu baca permission
- Pakai Pocket Universe extension
- Revoke approval setelah swap
