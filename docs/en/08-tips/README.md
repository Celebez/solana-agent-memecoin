# 💡 08. Tips from Experienced Traders & Anti-Scam

> 🇬🇧 English | 🇮🇩 [Bahasa Indonesia](../../id/08-tips/README.md)

> 🎯 **"The goal is not to get rich quick. The goal is to not lose money while learning."** — senior trader wisdom

## 🧠 Mindset (Most Important)

### 1. Crypto is Not a Shortcut to Riches
- 90% of traders **lose money** in their first 6 months
- Takes 1–2 years to become consistently profitable
- Treat it as **learning**, not "making money quick"

### 2. The 24-Hour Rule
When you feel FOMO wanting to buy, **wait 24 hours**. If you still feel you need it, then trade with small capital.

### 3. Journaling
Record every trade:
- Entry reason
- Exit reason
- P&L
- Lesson learned

Example template:
```markdown
### Trade 2026-07-05
- Token: $PEPE
- Entry: $0.00001 (MC $50K)
- Exit: $0.00002 (MC $100K)
- P&L: +95%
- Entry reason: trending on Twitter, holders rising
- Exit reason: take profit at 2x
- Lesson: timing matters, earlier exit is OK
```

### 4. Risk Management > Profit
Successful traders aren't the most profitable, but those who **rarely lose big**.

## 🚫 Fatal Mistakes to Avoid

### ❌ 1. All-In on One Token
```
Capital: $1000 → all-in $TOKEN_X → rug → $0
```
✅ Solution: max 5% per token, diversify across 10+ tokens.

### ❌ 2. Revenge Trading
```
Lost $200 → emotions → all-in another token → lost again → $0
```
✅ Solution: after a big loss, **stop trading 24–48 hours**.

### ❌ 3. FOMO at the Peak
```
Token up 100x → "might go 10x more" → buy → dump 90%
```
✅ Solution: if you just heard about the token, **you're already too late**.

### ❌ 4. Trust "Insider Info"
```
Telegram "alpha" group says token X will pump 1000x → buy → rug
```
✅ Solution: insider tips on Telegram = **99% scam**. Even if real, you become **exit liquidity**.

### ❌ 5. No Stop Loss
```
"This token will definitely go up again" → -50% → -80% → -99%
```
✅ Solution: stop loss **mandatory**, use mental stop (write in journal).

### ❌ 6. Borrow Money to Trade
```
Borrow $10K → trade → lose → debt → stress
```
✅ Solution: only trade money you're ready to lose 100%.

## 🕵️ Common Scam Schemes

### 1. 🚨 Rug Pull (Pull the Carpet)
```
Dev launches token → many buy → MC rises → dev sells all → MC → 0
```
**Avoid:**
- Check LP lock
- Avoid tokens without audit
- Monitor dev wallet (if dev sells, you sell too)

### 2. 🍯 Honeypot
```
Can buy token → can't sell (logic in smart contract)
```
**Avoid:**
- Test buy & sell small first
- Check if blacklist function exists
- See honeypot.is

### 3. 🐋 Whale Dump
```
1 whale holds 30% supply → sells slowly → price drops
```
**Avoid:**
- Check top holders concentration
- Avoid tokens with top 10 > 50%

### 4. 💸 High Buy Tax / Sell Tax
```
Buy 5% tax, sell 99% tax → dev can set tax arbitrarily
```
**Avoid:**
- Check tax on DexScreener or Birdeye before buying
- Normal: 0–5% tax. Dangerous: > 10%

### 5. 🎭 Fake Partnerships
```
"Will list on Binance tomorrow" → no listing → dump
```
**Avoid:**
- Check official announcements on exchange website
- Don't trust rumors without evidence

### 6. 🤖 Pump & Dump Group
```
1000 member group → "buy now!" → 1 person already bought first → dump on members
```
**Avoid:**
- If group is full of strangers, **you're the exit liquidity**
- Smart money doesn't need to share alpha

### 7. 🐛 Approve Exploit
```
Connect wallet → approve unlimited spend → drain all your tokens
```
**Avoid:**
- Always read permissions
- Use Pocket Universe extension
- Revoke approval after swap
